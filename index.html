Список команд Git

  • git clone - клонування репозиторію до комп'ютера
  • git status - показати стан файлів у робочому каталозі та індексі
  • git add - додавання вмісту робочого каталогу до індексу
  • git commit - збереження поточних версій файлів у локальний клон репозиторію
  • git push - завантаження локальних змін файлів на віддалений репозиторій
  • git checkout - створення та/або перемикання гілки репозиторію
